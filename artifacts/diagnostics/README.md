# Diagnostic Artifacts

Supporting material for [`docs/graphsage_diagnosis.md`](../docs/graphsage_diagnosis.md), [`docs/rgcn_diagnosis.md`](../docs/rgcn_diagnosis.md), and [`docs/fusion_diagnosis.md`](../docs/fusion_diagnosis.md).

| Subdirectory | Contents |
|--------------|----------|
| `gnn/plots/` | GraphSAGE training curves and HPO factor comparisons |
| `gnn/experiments.jsonl` | GraphSAGE epoch-level training log |
| `rgcn/plots/` | R-GCN training curves |
| `rgcn/ablations/` | Fold-0 relation ablations (full, no_pp, no_treats, no_bills) |
| `rgcn/experiments.jsonl` | R-GCN epoch-level training log |
| `fusion/experiments.jsonl` | Fusion phase training log |

Not cited as primary benchmarks. See `artifacts/published/` for 5-fold results.
