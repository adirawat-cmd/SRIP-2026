# Publication Figures

| Format | Path | Use |
|--------|------|-----|
| PNG (300 dpi) | `figures/png/` | Word, online supplements |
| SVG | `figures/svg/` | Web, editable vector |
| PDF | `figures/pdf/` | LaTeX (`\includegraphics`), print |

See [`docs/figure_catalog.md`](../docs/figure_catalog.md) for captions and data provenance.

Regenerate:

```bash
python scripts/10_generate_publication_figures.py
```
